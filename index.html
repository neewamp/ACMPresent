  • Adding a new page to your site

    • There are three things that need to happen for a new page to be added to your template website. This hopefully is super simple, to find out how to do more complicated things take a look at the django tutorial listed in the helpful urls section.

    Here are some things you will need to do:

    1. First update the list of urls that your site recognizes

      2. This can be found under the directory gettingstarted in the file urls.py

      urlpatterns = [
    url(r'^$', hello.views.index, name='index'),
    url(r'^db', hello.views.db, name='db'),
    url(r'^admin/', include(admin.site.urls)),
    url(r'^presentation/', hello.views.pres, name='pres'),#this line
]

      The arguments to the url function are:

      1. A regex to match urls typed into the browser
      2. The function in views.py that will return the correct html response
      3. The name that templating functions in django will recognize

    2. Next create a new directory in the folder /hello/templates called presentation

      3. Copy an html file from my github.

    3. Update views.py

      4. Views.py is located in the hello directory that is in the root of your project.
